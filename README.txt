PlantUML plugin for Maven
  https://github.com/jeluard/maven-plantuml-plugin

To create the project: MavenExample
  mvn archetype:generate -DgroupId=top.lesiji.plantuml -DartifactId=MavenExample

To generate the doc in the MavenExample directory
  mvn clean com.github.jeluard:plantuml-maven-plugin:generate
